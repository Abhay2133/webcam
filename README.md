## üì¶ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Abhay2133/webcam.git
   cd webcam
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run local development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
